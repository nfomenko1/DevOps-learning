**Навигация и файлы**:
> pwd - Показывает полный путь текущей папки

> ls - список файлов
ls -l - Подробно (права, размер, дата)
ls -la - Все файлы + права (полный доступ)
ls -lh - Читаемые размеры (1.2G вместо 1234567).

> cd /test - абсолютный путь (полный путь от корня файловой системы)
cd test - относительный путь (относительно ТЕКУЩЕЙ папки)
cd - Домой (то же что ~)
"cd -" - Предыдущая папка (вернуться туда, откуда пришел)
cd .. - На одну папку назад (выйти из текущей папки на уровень выше)

> cat файл - Показать содержимое

> grep - Global Regular Expression Print (поиск)
> grep "текст" файл  # Поиск текста
> 
> Пример:
> grep "искомый_текст" файл 
> grep "текст" /var/log/syslog
> 
> **Ключевые флаги:**
> grep -i "error" file     # Игнорировать регистр (Error=error)
> grep -v "ok" file        # ИНВЕРСИЯ — показать строки БЕЗ "ok"  
> grep -n "error" file     # НОМЕР строки + текст
> grep -r "nikita" /etc/   # Рекурсивно ВСЕ файлы в /etc/
> grep -A 2 "error" file   # +2 строки ПОСЛЕ совпадения
> grep -B 2 "error" file   # +2 строки ДО совпадения

> `find` — Поиск файлов
> find (ПУТЬ) (КРИТЕРИЙ) (ДЕЙСТВИЕ)
> 
> find /tmp -name "*.txt" - Найти файлы
> 
> **Основные критерии:**
> find /tmp -name "test.txt"          # ТОЧНОЕ имя
> find /tmp -name "test*"             # test + что угодно
> find /home -type f                  # Только ФАЙЛЫ (не папки)
> find /home -type d                  # Только ПАПКИ
> find /var -size +10M                # Больше 10МБ
> find /etc -user root                # Владельца root
> find /tmp -mtime 1                  # Изменены за последние 24ч
> 
>  **Действия после find:**
> find /tmp -name "*.txt" -delete     # Удалить найденное
> find /etc -name "*.conf" -exec cat {} \;  # Выполнить cat на каждом
> find /tmp -name "test*" | xargs rm  # Удалить через xargs

 `chmod` — Change Mode (ПРАВА ДОСТУПА)**
 
 **Linux права — 3 группы × 3 права:**
 rwx rwx rwx
│││ │││ │││
│││ │││ └┬┬┬─ Остальные (others)
│││ │└┬┬── Группа (group)  
│││ └┬┬──── Владелец (owner)
││└──┬────── x=исполнение
│└────┬───── w=запись
└──────┬──── r=чтение

 **Числовой формат (octal):**
 r=4  w=2  x=1
└───┴───┴─── Сложить!

755 = 7(rwx) 5(r-x) 5(r-x)    # Скрипт: владелец полный, остальные чтение+исп
644 = 6(rw-) 4(r--) 4(r--)    # Файл: владелец rw, остальные r
777 = 7 7 7                    # Полный доступ всем (ОПАСНО!)
