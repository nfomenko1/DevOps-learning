1. В общей папке /mnt/shared создана папка web-site, в которой лежит файл app.py, в котором будут проводится дальнейшие действия по созданию сайта.

2. После переустановки линукса заново установил pip/flask:

Устанавливаем pip (обновляем список пакетов Ubuntu):
- sudo apt update 
- sudo apt install python3-pip -y - 

**Разбор:**
- `sudo` = права root
- `apt install` = установить пакет
- `python3-pip` = pip для Python 3
- `-y` = "да всем вопросам"

Устанавливаем Flask:
cd /mnt/shared/web-site
pip3 install --

pip3 install --user flask - 

Разбор:
- `pip3` = менеджер пакетов Python 3
- `install` = установить
- `--user` = только для моего пользователя  (`~/.local/lib/python3.12/site-packages/`)
- `flask` = веб-фреймворк

3. Запустил сайт:
cd /mnt/shared/web-site
python3 app.py

Сайт запустился

4. Тест работы сайта через терминал:
- curl http://localhost:5000 - выводит текст из первого блока. В терминале с запущенным сайтом фиксируются логи.
