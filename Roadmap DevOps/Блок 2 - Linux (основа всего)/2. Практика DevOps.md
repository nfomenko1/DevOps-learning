#!/bin/bash
echo "---Linux Check Script---"
echo "1. Текущий путь:"
pwd
echo "2. Содержимое папки с подробностями:"
ls -la
echo "3. Какие процессы работают:"
ps aux | head -5
echo "4. System logs (last 10 lines):"
tail -10 /var/log/syslog
echo "5. Проверка ошибок/событий системы:"
df -h
echo "---Check complete---"

> Что делает **каждая строка**:

**`#!/bin/bash`**  
**Что делает**: говорит системе "это bash-скрипт, запускай через bash".  
**Зачем**: без неё Linux не поймёт, как запустить файл.

**`echo "текст"`**  
**Что делает**: выводит текст на экран.  
**Зачем**: красиво оформляет вывод (заголовки).

**`pwd`**  
**Что делает**: показывает **полный путь** текущей папки.  
**Зачем**: проверяем, где мы находимся.

**`ls -la`**  
**Что делает**: показывает **все файлы** с подробностями.  
**Зачем**: проверяем содержимое папки.

**`ps aux | head -5`**  
**Что делает**: `ps aux` = все процессы, `|` = труба, `head -5` = первые 5 строк.  
**Зачем**: смотрим, какие процессы работают.

**`tail -10 /var/log/syslog`**  
**Что делает**: `tail -10` = последние 10 строк, `/var/log/syslog` = системный лог.  
**Зачем**: проверяем ошибки/события системы.

**`df -h`**  
**Что делает**: показывает **использование дисков** в человекочитаемом формате.  
**Зачем**: сколько места свободно.

5. "**sudo apt update**" и "**sudo apt upgrade -y**" - обновление системы
sudo = права root (админа) 
apt = менеджер пакетов Ubuntu 
update = обновить список доступных пакетов 
upgrade = обновить все пакеты до новых версий
-y = "yes" автоматически (без вопросов)

6. **ip addr show** - найти ip адрес
ip = утилита для работы с сетью
addr = показать IP адреса всех интерфейсов

7.  **sudo systemctl start ssh** - запуск сервера.
**sudo systemctl enable ssh** - Автозапуск при загрузке.
