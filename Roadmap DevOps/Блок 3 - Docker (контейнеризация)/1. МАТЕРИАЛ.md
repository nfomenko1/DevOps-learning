**Цель блока:** научиться упаковывать приложения в контейнеры и собирать multi‑container‑связки.

- Docker: images/containers/volumes/networks, команды `docker run/pull/ps/rm`.
    
- Dockerfile, Docker Compose.

**Практика DevOps:**

- Папка `03-docker`.
    
- Dockerfile для PostgreSQL + простого API.
    
- `docker-compose.yml` для нескольких контейнеров (API + Postgres).​
    

**Практика сайта:**

- В папке `web-site` написать `Dockerfile` для Flask‑приложения:
    
    - базовый образ Python;
        
    - установка зависимостей;
        
    - запуск `app.py`.​
        
- Собрать и запустить:
    
    - `docker build -t devops-site ./web-site`
        
    - `docker run -p 5000:5000 devops-site`
        
- Проверить `curl http://127.0.0.1:5000` — теперь уже к контейнеру.
