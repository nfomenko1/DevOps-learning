**Цель блока:** понять базовые сети и git, собрать первый рабочий скрипт и минимальный сайт.

- Сети: OSI, TCP/UDP, IP/DNS/MAC, команды `nslookup`, `ping`, `curl`.
    
- Git: `init/clone/commit/branch/push`, GitHub‑репозиторий `devops-learning`.


**Практика DevOps:**

- Папка `01-networks`.
    
- Скрипт `check-network.sh`:
    
    - `nslookup google.com`
        
    - `ping 8.8.8.8` (2 пакета)
        
    - `curl https://httpbin.org/ip`
        
- README с описанием тестов.
    
- Первый коммит и `git push` на GitHub.

**Практика сайта:**

- В `devops-learning` создаём папку `web-site`.
    
- Минимальный Flask‑приложение `app.py`, которое возвращает страницу с твоим ФИО и одной кнопкой (пока без логики).​
    
- Проверка через `curl http://127.0.0.1:5000`.
