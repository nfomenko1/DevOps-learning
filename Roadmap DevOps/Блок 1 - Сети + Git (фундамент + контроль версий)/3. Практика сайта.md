1. **cd web-site** - Создали папку web-stie.
2. **touch app.py** - Создали файл app.py, в котором будет код для сайта.
3. **pip install flask** - установили flask

`pip` — менеджер пакетов Python. Скачивает Flask из интернета и устанавливает.  
**Зачем**: без Flask код `from flask import Flask` не заработает.

4. Добавили код в файл app.py:
![[Pasted image 20251229124614.png]]
Разбор кода:
1 строка: 
**Что делает**: импортирует основной класс `Flask` из библиотеки.  
**Зачем**: без него не запустится веб-сервер.

3 строка:
**Что делает**: создаёт объект веб-приложения. `__name__` — имя текущего модуля (файла).  
**Зачем**: Flask привязывает все маршруты к этому объекту `app`.

5 строка:
**Что делает**: декоратор — говорит Flask: «когда браузер зайдёт на адрес `/` (главная страница), выполни функцию ниже».  
**Зачем**: определяет **маршрут** — связь адрес → функция.

6-7 строка:
**Что делает**: функция `hello()` возвращает HTML-код страницы.  
**Зачем**: это **контент**, который увидит браузер.

9-10 строка:
**Что делает**:
- `if __name__ == '__main__'` — запускать только если файл запущен напрямую (`python app.py`), а не импортирован.
    
- `app.run(...)` — запускает сервер на порту 5000.
    
    - `debug=True` — автоматическая перезагрузка при изменении кода.
        
    - `host='0.0.0.0'` — доступен не только localhost, но и с других машин.  
        **Зачем**: **старт веб-сервера**.

5. **python app.py** - запустили веб-сервер и вывели надпись на http://localhost:5000.
**Что делает**: запускает веб-сервер на порту 5000.  
**Зачем**: проверить, что сайт отвечает на запросы.
