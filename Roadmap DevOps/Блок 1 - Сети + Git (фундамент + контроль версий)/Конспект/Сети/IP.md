**IP-адрес (Internet Protocol address)** — это уникальный числовой идентификатор устройства, подключённого к сети интернет или локальной сети.

> **Динамический** - адрес, который выдаётся на время (аренду) и может меняться.
- Может оставаться тем же при переподключении, если срок аренды не истёк и адрес свободен.
- Может поменяться при новом подключении, смене точки доступа, перезагрузке роутера или по решению провайдера.

> **Статический** - адрес, который закреплён за устройством и не меняется сам по себе
- Один и тот же сегодня, завтра, через месяц.
- Обычно даётся по отдельному тарифу/настройке (у провайдера или в корпоративной сети).

> **Белый (публичный) IP** — это адрес, который существует в глобальном интернете и уникален там в каждый момент времени.​  
Если у устройства есть белый IP, к нему можно подключиться извне напрямую: поднять сайт, VPN, RDP, игровой сервер и т.п.

**По‑простому**: 
это как реальный адрес здания в городе — любой человек, зная его, может приехать прямо к твоему дому.
Если этот IP адрес закреплен только за одни пользователем и ни за кем либо другим, то это белый IP адрес

> **Серый (частный) IP** — это адрес, который используется только внутри локальной сети: дома, в офисе, в сети провайдера.
У разных сетей могут быть одинаковые серые IP (например, у тебя дома и в офисе может быть 192.168.0.10), и это не конфликтует, потому что они «живут» в разных локальных мирах

**По‑простому**: 
это как номер квартиры внутри дома — квартир «12» много по стране, но внутри одного подъезда номер уникален.
Если этот IP используют множество пользователей, то есть делят между собой, то это серый IP адрес
