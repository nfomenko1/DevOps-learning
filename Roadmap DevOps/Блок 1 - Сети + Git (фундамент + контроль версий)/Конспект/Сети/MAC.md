**MAC‑адрес (Media Access Control)** — это 48‑битный номер, обычно записанный как шесть пар символов: AA:BB:CC:11:22:33.

Первые 3 пары обычно относятся к производителю (например, Intel, TP‑Link).

Последние 3 пары — уникальный номер конкретной сетевой карты.

**Важный момент**: один компьютер может иметь несколько MAC‑адресов — по одному на каждый интерфейс (Ethernet, Wi‑Fi, виртуальные адаптеры и т.д.).

**По простому**: IP адрес используется для определения местоположения устройства, MAC адрес уже конкретно идентифицирует его.
