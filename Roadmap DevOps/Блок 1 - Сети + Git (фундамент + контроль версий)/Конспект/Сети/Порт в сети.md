**Порт в сети** — это условный номер «двери» внутри одного IP‑адреса, через которую общается конкретное приложение или сервис.

Был добавлен для использования одинакового айпи с разными портами.

Порт — это число от 0 до 65535, которое записывается в заголовке TCP/UDP и вместе с IP‑адресом образует «конечную точку» соединения.  

Комбинация **IP + порт** однозначно определяет, какому сервису на хосте предназначены данные (например, 192.0.2.1:80 — веб‑сервер, 192.0.2.1:22 — SSH).

**Типы портов (по диапазонам):**
- 0–1023 — **well‑known**, стандартные: 80 (HTTP), 443 (HTTPS), 22 (SSH), 25 (SMTP), 53 (DNS).
- 1024–49151 — **зарегистрированные**: используют разные приложения/службы.
- 49152–65535 — **динамические/эпhemeral**: обычно временные порты клиента при исходящих соединениях (когда твой ПК идёт на чужой сервер).
