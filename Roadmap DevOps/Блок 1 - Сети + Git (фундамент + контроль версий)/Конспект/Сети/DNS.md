**DNS** — система, которая по доменному имени возвращает IP‑адрес сервера.

**По‑простому**: 
телефонная книга интернета — ты знаешь «имя» (google.com), а DNS говорит «номер телефона» (IP).

**Важно**: Если ранее сайт посещался, то запись останется в кеше, чтобы в последующие разы не тратить время на поиск IP адреса
