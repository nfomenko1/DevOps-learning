## Что делает `git init`

`git init` превращает **текущую папку** в git‑репозиторий.[](https://docs.altlinux.org/ru-RU/archive/7.0.5/html/simply-linux/ch39s02s03.html)​

Конкретно он:

- создаёт скрытую папку `.git` внутри текущего каталога;
    
- включает отслеживание изменений во **всех файлах и подпапках** этой папки;
    
- дальше все `git add/commit/push` относятся к этому репозиторию.



































## Дополнительные команды:
- `mkdir` — create directory, создать папку.​
    
- `-p` — «создавай все недостающие родительские папки и не ошибайся, если уже существуют».

`cd` — change directory, перейти в указанную папку.
`pwd` — print working directory, показать текущий путь.
`ls` — list, показать файлы и папки в текущей директории.

- `git` — команда системы контроля версий Git.
    
- `config` — настройка параметров Git.
    
- `--global` — применить настройку **для всех репозиториев** на этом компьютере.
    
- `user.name` — имя автора коммитов (то, что будет видно в истории).
- `user.email` — почта автора коммитов (обычно та же, что на GitHub).


- `nano` — простой текстовый редактор в терминале.
    
- `check-network.sh` — имя файла скрипта (расширение `.sh` = shell script).
