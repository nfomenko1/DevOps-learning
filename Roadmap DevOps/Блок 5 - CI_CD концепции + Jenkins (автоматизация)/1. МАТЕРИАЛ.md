**Цель блока:** автоматизировать сборку и деплой через pipeline.​

- Понятия CI/CD, stages: build / test / deploy.
    
- Jenkins как CI‑сервер.
    

**Практика DevOps:**

- Папка `05-cicd`.
    
- Установить Jenkins в Docker.
    
- Написать Jenkinsfile для Dockerized API+Postgres.​
    

**Практика сайта:**

- Добавить в Jenkins pipeline для сайта `web-site`:
    
    - берёт код из GitHub;
        
    - выполняет `docker build` образа `devops-site`;
        
    - делает `docker push` в Docker Hub/локальный registry;
        
    - запускается на каждый коммит в `main`.
