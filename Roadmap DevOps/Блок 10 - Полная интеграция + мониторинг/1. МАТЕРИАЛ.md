**Цель блока:** собрать всё в один рабочий DevOps‑pipeline и довести сайт до «продового» состояния.​

- Полная цепочка: Git commit → Jenkins CI → Docker registry → ArgoCD → K8s → мониторинг (Prometheus/Grafana).​
    

**Практика DevOps:**

- Папка `10-full-pipeline`.
    
- Скрипты/манифесты для полного развёртывания кластера и сервисов.
    
- Настройка метрик и дашбордов в Grafana.​
    

**Практика сайта:**

- Рефакторинг проекта `web-site`:
    
    - нормальная структура Flask‑приложения (blueprints, разделение слоёв);
        
    - новый дизайн/функционал (страницы портфолио, API и т.п.);
        
    - сбор метрик HTTP/ошибок в Prometheus и отображение их в Grafana.
